# 猩球价格表网站复刻项目

本项目是一比一复刻的猩球价格表网站，包含主菜单页面和所有子菜单页面。

## 项目结构

```
1-10/
├── index.html              # 主菜单页面
├── css/
│   ├── style.css          # 主样式文件
│   └── subpage.css        # 子页面样式文件
├── images/                # 图片资源文件夹
│   ├── main/              # 主页面图片
│   ├── 端游/              # 端游相关图片
│   ├── 手游/              # 手游相关图片
│   ├── vip/               # VIP相关图片
│   └── 护航单/            # 护航单相关图片
├── pages/                 # 子页面文件夹
│   ├── 端游/              # 端游子页面（8个）
│   ├── 手游/              # 手游子页面（8个）
│   └── vip/               # VIP子页面（2个）
├── download_images.py     # 图片下载脚本
└── create_all_pages.py    # 批量创建页面脚本
```

## 使用方法

### 1. 下载图片资源

由于图片URL有时效性签名，需要从原网站获取最新的图片URL。

**方法一：使用提供的下载脚本（需要更新URL）**
```bash
python3 download_images.py
```

**方法二：手动下载**
1. 访问原网站：https://app.lilihuyu.com/s/pages/7000000003731339
2. 使用浏览器开发者工具提取图片URL
3. 将图片下载到对应的 `images/` 文件夹中

### 2. 查看网站

直接在浏览器中打开 `index.html` 文件即可查看主菜单页面。

## 图片文件夹说明

所有图片都按类别分类存储在对应的文件夹中：

- `images/main/` - 主页面使用的图片（头部logo、背景图等）
- `images/端游/` - 端游相关图片（菜单图标、内容图片）
- `images/手游/` - 手游相关图片（菜单图标、内容图片）
- `images/vip/` - VIP相关图片（菜单图标、内容图片）
- `images/护航单/` - 护航单权益相关图片

## 注意事项

1. **图片资源**：由于原网站的图片URL包含时效性签名，提供的下载脚本中的URL可能会过期。需要从原网站重新获取最新的图片URL。

2. **子页面内容**：部分子页面可能需要访问实际页面来获取完整的内容图片。当前创建的子页面使用了占位图片路径。

3. **样式适配**：CSS样式已经尽可能复刻原网站的视觉效果，但可能需要根据实际图片尺寸进行微调。
